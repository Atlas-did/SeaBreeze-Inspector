## 二、接线图

```
Arduino Nano          PCA9685              SG90舵机×3
┌─────────┐          ┌────────┐
│         │          │        │         ┌───┐
│ 5V  ────┼──────────┤VCC     │         │S1 │── 底座 (theta1) → D0
│ GND ────┼──────────┤GND     │         └───┘
│ A4  ────┼──(SDA)───┤SCL     │         ┌───┐
│ A5  ────┼──(SCL)───┤SDA     │         │S2 │── 大臂 (theta2) → D1
│         │          │        │         └───┘
│ TX  ────┼──(蓝牙)   │        │         ┌───┐
│ RX  ────┼──(蓝牙)   │        │         │S3 │── 小臂 (theta3) → D2
└─────────┘          └──┬─────┘         └───┘
                        │
                    PWM输出(D0-D2)

注意:
  - PCA9685 外接 5V 电源 (舵机电流大, Arduino 5V 不够)
  - HC-06 VCC 接 Arduino 3.3V (不要接 5V!)
  - 所有设备共地 (GND 连接在一起)
```

## 三、组装步骤

### Step 1: 机架安装
1. 将亚克力/3D打印底板安装到 Tello 底部
2. 用 M3 螺丝固定, 确保不遮挡摄像头和下视传感器

### Step 2: 舵机安装
1. **底座舵机(S1)**: 水平安装, 旋转轴朝上
2. **大臂舵机(S2)**: 垂直安装, 固定到底座舵机臂上
3. **小臂舵机(S3)**: 垂直安装, 固定到大臂末端
4. 连杆用舵机附带的塑料臂 + 热熔胶固定

### Step 3: 电路连接
1. PCA9685 → Arduino: VCC→5V, GND→GND, SCL→A5, SDA→A4
2. 舵机 → PCA9685: S1→D0, S2→D1, S3→D2
3. PCA9685 外接 5V 电源 (USB充电宝即可)
4. HC-06 → Arduino: VCC→3.3V, GND→GND, TX→RX, RX→TX

### Step 4: 固件烧入
```bash
# 见 docs/FLASH_GUIDE.md
python scripts/flash_firmware.py
```

## 四、注意事项

1. **重心**: 机械臂总重应 <100g, 否则影响飞行稳定
2. **电源**: PCA9685必须外接电源, Arduino 5V 无法驱动3个舵机
3. **蓝牙**: HC-06接3.3V, 接5V会烧毁
4. **安全**: 首次测试先拿掉螺旋桨保护罩, 低空(0.5m)测试
