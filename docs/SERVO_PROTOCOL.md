# 机械臂舵机控制器 — 通信协议文档

> 协议版本: v1.0 | 波特率: 115200 | 编码: ASCII

---

## 一、协议概述

采用**简单文本协议**，每条指令以**换行符 `\n`** 结束，天然帧边界。所有数据为ASCII可打印字符，可直接用Arduino IDE串口监视器调试。

| 属性 | 值 |
|------|-----|
| 波特率 | 115200 bps |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验 | 无 (None) |
| 流控 | 无 |
| 帧结束符 | `\n` (LF) 或 `\r\n` (CRLF) |
| 编码 | ASCII |

---

## 二、指令集

### 1. 绝对角度模式 (Absolute)

| 项目 | 内容 |
|------|------|
| **格式** | `A<base>,<shoulder>,<elbow>` |
| **参数** | base: 底座角度 0-180；shoulder: 大臂角度 0-180；elbow: 小臂角度 0-180 |
| **示例** | `A90,45,30` → 底座90°、大臂45°、小臂30° |
| **响应** | `[ABS] 目标角度 → Base:90 Shoulder:45 Elbow:30` |
| **错误** | 角度超出0-180范围时返回 `[ERR] 角度超出范围[0,180]: xxx` |

### 2. 相对增量模式 (Relative)

| 项目 | 内容 |
|------|------|
| **格式** | `R<delta_base>,<delta_shoulder>,<delta_elbow>` |
| **参数** | 三个增量值（可为负），在当前角度上增减 |
| **示例** | `R10,0,-5` → 底座+10°、大臂不变、小臂-5° |
| **响应** | `[REL] 增量 (10,0,-5) → 目标 Base:100 Shoulder:45 Elbow:25` |
| **说明** | 计算后的目标角度自动限制在0-180范围内 |

### 3. 归位 (Home)

| 项目 | 内容 |
|------|------|
| **格式** | `H` |
| **参数** | 无 |
| **示例** | `H` |
| **响应** | `[HOME] 正在归位到 (90, 90, 90)...` → `[OK] 运动完成` |
| **说明** | 平滑移动到90,90,90默认姿态 |

### 4. 查询角度 (Query)

| 项目 | 内容 |
|------|------|
| **格式** | `Q` |
| **参数** | 无 |
| **示例** | `Q` |
| **响应** | `A:90,S:45,E:30`（Base, Shoulder, Elbow） |
| **说明** | 返回当前实际角度，不是目标角度 |

### 5. 紧急停止 (Stop)

| 项目 | 内容 |
|------|------|
| **格式** | `S` |
| **参数** | 无 |
| **示例** | `S` |
| **响应** | `[STOP] 紧急停止，当前位置已锁定` |
| **说明** | 立即停止运动，目标角度设为当前角度 |

### 6. 帮助 (Help)

| 项目 | 内容 |
|------|------|
| **格式** | `?` |
| **参数** | 无 |
| **示例** | `?` |
| **响应** | 打印完整指令帮助信息 |

## 三、上电初始化流程

```
上电
  → 初始化串口 (115200)
  → 初始化PCA9685 (I2C, 0x40, 50Hz)
  → 打印版本信息
  → 【所有舵机归位到 90, 90, 90】
  → 启动看门狗计时器
  → 等待串口指令...
```

---

## 四、错误响应

| 错误码 | 触发条件 | 示例响应 |
|--------|---------|---------|
| 格式错误 | 指令参数数量不对 | `[ERR] 绝对模式格式错误` |
| 范围错误 | 角度不在0-180 | `[ERR] 角度超出范围[0,180]: 200` |
| 未知指令 | 首字符不在A/R/H/Q/S/?中 | `[ERR] 未知指令: X123` |

## 五、通信时序示例

```
PC发送:   A90,45,30\n
Nano响应: [ABS] 目标角度 → Base:90 Shoulder:45 Elbow:30
          [OK] 运动完成

PC发送:   Q\n
Nano响应: A:90,S:45,E:30

PC发送:   R10,0,-5\n
Nano响应: [REL] 增量 (10,0,-5) → 目标 Base:100 Shoulder:45 Elbow:25
          [OK] 运动完成

PC发送:   H\n
Nano响应: [HOME] 正在归位到 (90, 90, 90)...
          [OK] 运动完成
```

## 六、Python调用示例

```python
import serial
import time

# 连接Arduino（Windows用COM3，Linux用/dev/ttyUSB0）
ser = serial.Serial("COM3", 115200, timeout=1)
time.sleep(2)  # 等待Nano初始化完成

# 绝对角度模式
ser.write(b"A90,45,30\n")
print(ser.readline().decode().strip())

# 等待运动完成
ser.readline()

# 查询当前角度
ser.write(b"Q\n")
print(ser.readline().decode().strip())  # 输出: A:90,S:45,E:30

# 相对增量模式
ser.write(b"R10,0,-5\n")
print(ser.readline().decode().strip())

ser.close()
```
