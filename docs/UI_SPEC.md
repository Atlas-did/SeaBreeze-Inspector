# 监控面板 UI 设计说明

## 一、架构选型

| 方案 | 开发速度 | 美观度 | 跨平台 | 手机支持 | 学习成本 | 推荐 |
|------|---------|--------|--------|---------|---------|------|
| PyQt5 | 中 | 高 | 桌面 | 否 | 中 | ★★★ |
| **Web Dashboard** | **中** | **很高** | **是** | **是** | **中** | **★★★★** |
| Dear PyGui | 快 | 高 | 桌面 | 否 | 低 | ★★★ |
| Streamlit | 极快 | 中 | 是 | 是 | 极低 | ★★ |

**选择 Tkinter (Python内置)**：零依赖，开发最快，适合快速原型演示。后续可迁移到 PyQt5 或 Web。

## 三、控件信号对应表

| 控件 | 事件 | 回调函数 | 功能 |
|------|------|---------|------|
| [起飞] 按钮 | click | on_takeoff() | 调用 TelloController.takeoff() |
| [降落] 按钮 | click | on_land() | 调用 TelloController.land() |
| [悬停] 按钮 | click | on_hover() | 发送悬停指令 |
| [紧急停止] | click | on_emergency() | 调用 emergency() |
| 目标高度滑块 | change | on_target_height(val) | 更新目标 Z |
| 目标位置输入 | return | on_target_pos(x,y,z) | 更新目标点 |
| 关节角度输入 | return | on_joint_angles(t1,t2,t3) | 调用 IK + set_joint_angles |
| [亮/暗] 按钮 | click | toggle_theme() | 切换颜色主题 |
| [截图] 按钮 | click | on_screenshot() | 保存到 data/screenshots/ |

## 四、线程设计

```
主线程 (UI):
  - Tkinter 事件循环
  - 控件更新

工作线程 1 (Video):
  - 从 TelloVideoStream 获取帧
  - 调用 YOLO 检测
  - 绘制检测框
  - 每秒 30 帧更新到 Canvas

工作线程 2 (State):
  - 轮询 Tello 状态 (电池/高度/姿态)
  - 轮询机械臂状态 (关节角度)
  - 每 500ms 更新数据面板

工作线程 3 (3D Render):
  - 更新无人机/机械臂 3D 位置
  - 重绘俯视图
  - 每 100ms 更新
```

> 所有耗时操作在独立线程，禁止阻塞主线程。
