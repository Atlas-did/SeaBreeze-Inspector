# 引脚定义与接线图

> 本文档定义了Arduino Nano、PCA9685、舵机和外接电源之间的全部引脚对应关系。
> 接线时务必对照此表，接错可能导致设备损坏。

---

## 一、Arduino Nano (ATmega328P, CH340)

### 1.1 Nano引脚总览图

```
                    ┌─────────────────────────┐
    D13/SCK  (PC5) │●                        │●  D12/MISO (PC4)
    3.3V           │●                        │●  D11/MOSI/PWM (PB3)
    AREF           │●      Arduino Nano      │●  D10/SS/PWM   (PB2)
    A0/PC0         │●                        │●  D9/PWM        (PB1)
    A1/PC1         │●                        │●  D8            (PB0)
    A2/PC2         │●                        │●  D7            (PD7)
    A3/PC3         │●                        │●  D6/PWM        (PD6)
    A4/SDA (PC4)   │●  ┌──────┐   ┌──────┐  │●  D5/PWM        (PD5)
    A5/SCL (PC5)   │●  │CH340 │   │RESET │  │●  D4            (PD4)
    D0/RX  (PD0)   │●  │ USB  │   │ 按钮 │  │●  D3/PWM/INT1   (PD3)
    D1/TX  (PD1)   │●  └──────┘   └──────┘  │●  D2/INT0       (PD2)
    GND            │●                        │●  GND
    RESET          │●                        │●  5V
    VIN (7-12V)    │●                        │●  VIN (7-12V)
                    └─────────────────────────┘
                          Mini-USB口（朝下）
```

### 1.2 Nano引脚分配表

| Nano引脚 | 功能 | 连接至 | 说明 |
|:--------:|------|--------|------|
| A4 | **SDA** (I2C数据线) | PCA9685 SDA | 硬件I2C，固定引脚不可改 |
| A5 | **SCL** (I2C时钟线) | PCA9685 SCL | 硬件I2C，固定引脚不可改 |
| 5V | 5V电源输出 | PCA9685 VCC | Nano为PCA9685提供逻辑电源 |
| GND | 地线 | PCA9685 GND | 共地连接 |
| D0/D1 | RX/TX (串口) | USB→CH340→PC | 用于串口通信和烧入 |
| Mini-USB | USB数据+供电 | PC | 程序烧入 + Nano供电 + 串口通信 |

**未使用的引脚：** D2-D13, A0-A3, 3.3V, AREF, VIN, RESET
（为后续扩展预留：如超声波、额外传感器等）

## 二、PCA9685 舵机驱动板

### 2.1 PCA9685引脚说明

```
    ┌──────────────────────────────────┐
    │         PCA9685 模块             │
    │                                  │
    │  VCC  GND  SCL  SDA         OE  │  ← 左侧：I2C接口
    │   ●    ●    ●    ●          ●   │
    │                                  │
    │  PWM0 PWM1 PWM2 ... PWM15        │  ← 下方：16路PWM输出
    │   ●    ●    ●        ●          │
    │                                  │
    │  V+   GND (右侧电源接口)          │  ← 右侧：舵机动力电源
    │   ●    ●                         │
    │                                  │
    │  [A0 A1 A2 A3 A4 A5]            │  ← I2C地址选择焊盘
    └──────────────────────────────────┘
```

### 2.2 PCA9685引脚分配表

| PCA9685引脚 | 连接至 | 功能 | 颜色 |
|:-----------:|--------|------|------|
| **VCC** | Nano 5V | 逻辑电源 (3.3V-5V) | 红线 |
| **GND** | Nano GND | 逻辑地 | 黑/棕线 |
| **SCL** | Nano A5 | I2C时钟 | — |
| **SDA** | Nano A4 | I2C数据 | — |
| **V+** | **外接5V 2A电源+** | **舵机动力电源** | **红色粗线** |
| **GND** | **外接5V 2A电源-** | **舵机动力地** | **黑色粗线** |
| **PWM0** | S1底座舵机信号 | 底座旋转控制 | 黄线 |
| **PWM1** | S2大臂舵机信号 | 大臂俯仰控制 | 黄线 |
| **PWM2** | S3小臂舵机信号 | 小臂俯仰控制 | 黄线 |
| PWM3-PWM15 | 悬空 | 预留扩展 | — |
| **OE** | 悬空/接GND | 输出使能（拉低使能输出） | — |

### 2.3 I2C地址设置

PCA9685的默认I2C地址为 **0x40**（所有A0-A5焊盘未连接）。

如需修改地址，通过焊接A0-A5焊盘到VCC实现：

| A5 | A4 | A3 | A2 | A1 | A0 | I2C地址 |
|:--:|:--:|:--:|:--:|:--:|:--:|:-------:|
| GND| GND| GND| GND| GND| GND| **0x40** (默认) |
| GND| GND| GND| GND| GND| VCC| 0x41 |
| GND| GND| GND| GND| VCC| GND| 0x42 |
| ... | | | | | | 0x40-0x7F |

> ⚠️ 如果多块PCA9685共用I2C总线，必须通过焊接设置不同地址。

## 三、SG90舵机

### 3.1 SG90引脚定义

每颗SG90舵机有3根线：

| 线色 | 功能 | 连接至 | 说明 |
|:----:|------|--------|------|
| **红** | VCC (+5V) | **PCA9685 V+** (外接电源) | **严禁接Nano 5V！** |
| **棕/黑** | GND (地线) | **PCA9685 GND** (外接电源) | 共地 |
| **黄/橙** | Signal (PWM信号) | PCA9685 PWMx | 控制信号 |

### 3.2 舵机分配表

| 编号 | 名称 | 英文名 | PCA9685通道 | Nano内对应 | 功能 | 角度范围 |
|:----:|------|--------|:-----------:|------------|------|:--------:|
| S1 | 底座舵机 | Base Servo | **PWM0** | — | 机械臂水平旋转 (Yaw) | 0°-180° |
| S2 | 大臂舵机 | Shoulder Servo | **PWM1** | — | 大臂俯仰 (Pitch) | 15°-165° |
| S3 | 小臂舵机 | Elbow Servo | **PWM2** | — | 小臂俯仰 (Pitch) | 0°-180° |

## 四、外接电源接线

### 4.1 电源需求分析

| 负载 | 数量 | 典型电流 | 峰值电流 | 说明 |
|------|:----:|:--------:|:--------:|:----:|
| SG90舵机 | 3 | 100mA×3 | 250mA×3 | 运动瞬间电流最大 |
| PCA9685芯片 | 1 | 10mA | 25mA | 逻辑电路 |
| **合计** | — | **310mA** | **775mA** | — |

Nano USB供电能力：**最大500mA**（且部分用于Nano自身运行）

**结论：必须外接5V 2A独立电源！**

### 4.2 电源接线图

```
┌─────────────────────────────────────────────────────────────────────┐
│                        外接5V 2A电源模块                              │
│                                                                     │
│   ┌──────────┐          红线(V+, +5V)          ┌──────────┐        │
│   │          │◄────────────────────────────────►│ PCA9685  │        │
│   │ 5V 2A    │                                  │   V+     │        │
│   │ 电源适配器 │          黑线(GND)               │          │        │
│   │ (或电池)  │◄────────────────────────────────►│   GND    │        │
│   └──────────┘                                  └────┬─────┘        │
└───────────────────────────────────────────────────────┼──────────────┘
                                                        │
                              ┌─────────────────────────┘
                              │ 共地
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│                         Arduino Nano                                │
│                                                                     │
│   Mini-USB ◄───数据线───► PC (USB供电+串口通信)                     │
│                                                                     │
│   5V引脚 ─────红线──────► PCA9685 VCC (逻辑电源, 电流<30mA)         │
│   GND引脚 ────黑线──────► PCA9685 GND (共地)                        │
│   A4引脚 ─────SDA───────► PCA9685 SDA                               │
│   A5引脚 ─────SCL───────► PCA9685 SCL                               │
└─────────────────────────────────────────────────────────────────────┘

舵机电源线连接:
  S1红线 ──► PCA9685 V+      S1棕线 ──► PCA9685 GND      S1黄线 ──► PWM0
  S2红线 ──► PCA9685 V+      S2棕线 ──► PCA9685 GND      S2黄线 ──► PWM1
  S3红线 ──► PCA9685 V+      S3棕线 ──► PCA9685 GND      S3黄线 ──► PWM2
```

### 4.3 电源接线要点

| 要点 | 说明 |
|------|------|
| **Nano和舵机电源分离** | Nano由USB供电，舵机由外接5V供电，两者只在GND处共地 |
| **红线绝对不能接Nano 5V** | 3个舵机同时运动峰值电流750mA，远超Nano 500mA上限 |
| **共地是必须的** | PCA9685的GND必须与Nano的GND相连，否则I2C通信不稳定 |
| **电源线尽量短** | 舵机电源线长度<30cm，过长会导致电压跌落和舵机抖动 |
| **推荐加电容滤波** | V+和GND之间并联100μF电解电容+0.1μF陶瓷电容 |

## 五、完整接线速查表

### 按连接关系排序

| 从 (From) | 到 (To) | 线色/类型 | 用途 |
|-----------|---------|----------|------|
| Nano A4 | PCA9685 SDA | 杜邦线 | I2C数据 |
| Nano A5 | PCA9685 SCL | 杜邦线 | I2C时钟 |
| Nano 5V | PCA9685 VCC | 杜邦线(红) | PCA9685逻辑电源 |
| Nano GND | PCA9685 GND | 杜邦线(黑) | 共地 |
| 外接5V+ | PCA9685 V+ | 电源线(红) | 舵机动力电源 |
| 外接5V- | PCA9685 GND | 电源线(黑) | 舵机动力地 |
| PCA9685 PWM0 | S1黄线 | 杜邦线 | 底座舵机信号 |
| PCA9685 PWM1 | S2黄线 | 杜邦线 | 大臂舵机信号 |
| PCA9685 PWM2 | S3黄线 | 杜邦线 | 小臂舵机信号 |
| PCA9685 V+ | S1红线 | 电源线 | 底座舵机电源 |
| PCA9685 V+ | S2红线 | 电源线 | 大臂舵机电源 |
| PCA9685 V+ | S3红线 | 电源线 | 小臂舵机电源 |
| PCA9685 GND | S1棕线 | 电源线 | 底座舵机地 |
| PCA9685 GND | S2棕线 | 电源线 | 大臂舵机地 |
| PCA9685 GND | S3棕线 | 电源线 | 小臂舵机地 |
| Nano Mini-USB | PC USB | USB线 | Nano供电+烧入+串口 |

### 按功能模块排序

| 功能 | 所需接线 | 数量 |
|------|---------|:----:|
| I2C通信 | SDA(A4)+SCL(A5)+VCC(5V)+GND | 4根杜邦线 |
| 舵机信号 | PWM0+PWM1+PWM2 → S1/S2/S3黄线 | 3根杜邦线 |
| 舵机动力 | 外接5V+ → V+ → 3根红线 | 1路电源分3路 |
| 共地 | Nano GND ↔ PCA9685 GND ↔ 外接5V- | 3处连接 |

## 六、接线检查清单

上电前逐项检查：

- [ ] Nano A4 → PCA9685 SDA 已连接
- [ ] Nano A5 → PCA9685 SCL 已连接
- [ ] Nano 5V → PCA9685 VCC 已连接
- [ ] Nano GND → PCA9685 GND 已连接（共地）
- [ ] 外接5V+ → PCA9685 V+ 已连接
- [ ] 外接5V- → PCA9685 GND 已连接
- [ ] S1(黄) → PCA9685 PWM0
- [ ] S2(黄) → PCA9685 PWM1
- [ ] S3(黄) → PCA9685 PWM2
- [ ] S1/S2/S3红线 → PCA9685 V+（不是Nano 5V！）
- [ ] S1/S2/S3棕线 → PCA9685 GND
- [ ] **没有短路**：所有裸露导线无接触
- [ ] **外接电源已关闭**：上电前确认电源开关在OFF

上电顺序：
1. 先接外接5V电源到PCA9685，**不开启**
2. 再接Nano的USB到电脑
3. 先开启外接5V电源
4. 再插Nano的USB（或按Nano复位键）
